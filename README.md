Yeets stuff up the stack

```javascript
const yeet = require("node-yeet");

let thisBitch = "stacy";
yeet(thisBitch);
```

For TypeScript:

```typescript
const yeet = import("node-yeet");
```

(or, with `--esModuleInterop`):

```typescript
import yeet from "node-yeet";
```

## How To Install?

```console
$ npm install --save node-yeet
```

Or, if you prefer Yarn:

```console
$ yarn add node-yeet
```
